---

> **注意**: 
> - `n` = Normal 模式, `i` = Insert 模式, `v` = Visual 模式, `x` = Visual/Select 模式, `s` = Select 模式, `c` = Command 模式
> - 這些快捷鍵基於您當前的 LazyVim 配置，實際可用的快捷鍵可能會根據已安裝的插件而有所不同
> - 使用 `<Space>sk` 在 Neovim 中查看當前可用的所有快捷鍵
> - 使用 `<Space>?` 查看當前緩衝區特定的快捷鍵
